## 2026-03-28 — Sprint 3: 인벤토리 그룹 뷰 안정화 + 성능 개선 + UX 정렬 (INV-12)

아이템 카드 확장 패널(INV-07)은 UX 복잡도 피드백으로 미채택, 기존 단순 `ConsumerWidget` 유지.
대시보드 → 필터 이동 시 묶음 카드 기본 접힘 + 발송-검수 정렬 고정 추가.

### 변경 파일 및 내용

| 파일 | 변경 내용 |
|------|-----------|
| `inventory/widgets/grouped_list_view.dart` | `SliverPersistentHeader` 제거 → `ListView.builder + GroupCard(ExpansionTile)` 전환. `loadBatchData` 상품 조회를 순차 `await` 루프 → `Future.wait` 병렬 처리로 변경(판매중 필터 딜레이 개선). `GroupCard`: `initiallyExpanded: false` (대시보드 이동 시 기본 접힘). 발송-검수 그룹 정렬에 `forceAsc: true` 적용(오래된 발송일 우선 고정). `_GroupHeaderDelegate`, `_expandedGroups`, `_initialized` 제거. |
